# INDIS 2026 - Conference Website

International Conference on Design & Innovation Studies - A responsive website built from Figma design.

## Project Structure

```
indis2026/
├── index.html          # Main HTML file
├── css/
│   └── style.css       # All styles
├── js/
│   └── main.js         # JavaScript functionality
├── images/
│   ├── *.svg           # Vector icons (included)
│   └── *.png/jpg       # Image placeholders (need to be added)
└── README.md           # This file
```

## Required Images (Placeholders)

The following images need to be added to the `images/` folder:

### Hero Section
- `hero-bg.jpg` - Background image (1920x1080px) - aerial view of garden/campus

### Logos
- `iitg-logo.png` - IIT Guwahati logo (340x~220px)
- `ld-logo.png` - LD Logo (~165x58px)
- `partner-logo.png` - Partner organization logo
- `sponsor-logo.png` - Sponsor logo (~285x86px)

## Features

- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Smooth Scrolling**: Navigation links scroll smoothly to sections
- **Side Menu**: Hamburger menu with smooth animation
- **Scroll Animations**: Elements animate in as you scroll
- **Jury Slider**: Interactive carousel for jury members
- **Parallax Effect**: Hero section has subtle parallax scrolling

## Sections

1. **Hero Section** - Conference branding and date
2. **Timeline** - Event schedule visualization
3. **Overview** - Conference description
4. **Call for Papers** - Research topics grid
5. **Important Dates** - Key dates cards
6. **Organisation** - Committee information
7. **Jury Members** - Jury member profiles
8. **Footer** - Contact and social links

## How to Use

1. Open `index.html` in a web browser
2. Add placeholder images to the `images/` folder
3. Customize content as needed
